### QuackTextFieldColors

꽥꽥의 TextField 관련 컴포넌트에 사용할 테마를 지정합니다. `QuackTheme`를 통해 꽥꽥 테마로 제공됩니다.

```kotlin
@Immutable
public interface QuackTextFieldColors
```

#### Properties

##### `cursor`

꽥꽥 컴포넌트에서 사용할 커서 색상

기본값으로 `QuackColor.Black`를 사용합니다.

```kotlin
public val cursor: QuackColor
```

### QuackColor

덕키에서 사용할 색상을 정의합니다.

색상 정의는 코드 스타일 통일을 위해 ARGB 형식이 아닌 Hex 형식으로 해야 합니다.

```kotlin
@Immutable
@JvmInline
public value class QuackColor(public val value: Color)
```

#### Constructor

| Field name |                 Description                 |
|:----------:|:-------------------------------------------:|
|  `value`   | 색상 값. 이 인자로 색상을 사용하기 위해 컴포즈의 `Color`로 받습니다. |

#### Properties

|      Name      |                         Preview                         |

|    `Black`     | <ColorBox width="100px" height="50px" color="#222222"/> |
